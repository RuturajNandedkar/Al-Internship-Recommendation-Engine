# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-26

### Added
- Core AI Engine for internship recommendations using Gemini AI.
- Express.js backend with MongoDB integration.
- Resume parsing via BullMQ background jobs.
- Redis caching for recommendation results.
- Comprehensive test suite for both frontend and backend.
- CI/CD pipelines via GitHub Actions.
- ESLint and Prettier configuration for code quality.
- Standardized documentation (CONTRIBUTING, PR Template).
